#### Steps to get an API key for IBM Watson Translator:

##Create an instance of the translation service on IBM Cloud:

- Go to the [Language Translator](https://console.bluemix.net/catalog/services/language-translator) page in the IBM Cloud Catalog.
- Sign up for a free IBM Cloud account or log in if you already have one.
- Select the free lite plan for testing the service.
- Click Create.


##Copy the credentials:

- On the service dashboard, click the Service credentials tab.
- Click View credentials under Actions.
- Copy the apikey, and url values.

##Use your key in the translation script:
- change the vlaue of the variable *iam_api_key* to your api key
